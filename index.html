BIN2DCO: HMXB+NSBH¶

This is the sketch of the plan for a two-folded study using detail binary evolution models. As a first goal, we will try to set new constraints in the different parameters that leads an initial massive binary to become a high-mass X-ray binary (HMXB) which contains a neutron star (NS) as the compact object accretor. This study includes a complete understading of the starting conditions (masses and orbital periods) and also the formation of the NS accompanied by an asymmetric natal-kick. This study would allow a direct contrast with observational data.

On the second part of the work, we will continue the evolution of such HMXB until the formation of a population of double compact object binaries (with two NS, or a black hole, BH). The aim in this case will be connected to the understanding of the low-mass region of the 'stellar graveyard' where there is a lack of compact objects with masses between 3-5 M$_\odot$. We then would be able to map initial-to-final parameters and try to estimate detectable rates with a net of all the gravitational waves detectors that are searching for these events.

On this presentation¶

In the following, we will outline the steps we are going to take in order to achieve the goals we have set. We first introduce the initial parameters that defines the grid we are going to explore with detail modelling. When the first compact object is formed, we will also show how we plan to add the asymmetric natal kick using a deterministic approach on top of the statistical nature of the highly-unknown natal kick.

def number_of_points(minim, maxim, step):
    '''evaluate the number of points an array in np.linspace will have'''
    return int(1 + (maxim - minim) / step)

def get_3Dgrid(pm_min, pm_max, n_pm, q_min, q_max, dqs, porb_min, porb_max, n_porb):
    '''create the 3D grid of binary parameters'''
    n_pm = int(n_pm); n_porb = int(n_porb)
    pms = np.power(10, np.linspace(np.log10(pm_min), np.log10(pm_max), num=n_pm))
    qs = np.arange(start=q_min, stop=q_max+0.001, step=dqs)
    porbs = np.power(10, np.linspace(np.log10(porb_min), np.log10(porb_max), num=n_porb))
    return pms, qs, porbs

1. Initial binary grid¶

This grid is composed of three parameters: initial masses (primary mass and mass-ratio) and the initial orbital period. Given the nature of the distribution of these parameters coming from observations (e.g., Sana et al. 2012, Moe & Di Stefano 2016), the steps are either linear (mass-ratios, dqs) or logarithmic (primary masses, dlpms, and orbital periods, dlps). We will also consider that they are independent between them so that we can independently sample from each of them:

3D grid of parameters¶

  • for the initial primary masses we use a logarithmic step to sample the range between 10-150 M$_\odot$, we choose dlpms = 0.05,

  • for the initial mass-ratios we sample in the range between 0.10-0.95, with a linear step of dqs = 0.05

  • for the initial orbital periods we also use a logarithmic step and sample in the range between 3-3000 days, with dlps = 0.10

Grid information¶

Exploring this 3D space in an extremely detailed manner is practically impossible due to the time spent in computing each of the binary evolution models. This is why we decide to create 2 grids for the same initial parameter limits: one in which the steps are small enough to have a very good coverage of the grid (dense grid) and other in which each of the steps is twice as big (sparse grid).

Thanks to this approach we will compute less models while still having a good statistical sample and if it is necessary, we will backup to the more detailed grid in regions where we need.

2. Core collapse stage¶

Once one of the stars in the binary reaches core-collapse, we will apply an asymmetric kick in order to constraint this unknown event.

This exploration will contain the following assumptions:

  • The distribution of natal kick magnitudes are drawn from a maxwellian distribution with a root mean square of 265 km s$^{-1}$ but lowering its value according to the fallback fraction ($f_{\rm b}$) of material that ends up forming the compact object: $v_{\rm kick} = (1 - f_{\rm b}) v_{\rm Maxwell}$

  • The post-kick parameters of orbital periods ($P_{\rm orb}$) and eccentricities ($e$) used for the continuation of the binary evolutions are bound to be between the 0.05-0.95 quantiles for $P_{\rm orb}$ and 0.0-1.0 quantiles for $e$

  • The grid of post-kick parameters is divided between 10 bins in $P_{\rm orb}$ and 10 bins in $e$

2.1 Exemplary case of post-kick orbital parameters¶

We show an example on how the grid will be divided after the post-kick.

It contains a collapsing star of 12.81 M$_\odot$ forming a BH of 8 M$_\odot$, orbiting around a companion star of 7 M$_\odot$. The fraction of matter falling back to the BH is $f_{\rm b} = 0.5$

Show Code
# create BinarySystem object
binary = BinarySystem(m1=12.81, m1_core_mass=9, m1_remnant_mass=8, m1_fallback_fraction=0.50, m2=7, P=5)

# set the distribution of random kicks. For that, we use a 50000 samples from a Maxwellian distribution scaled
# by the fallback fraction
binary.set_natal_kick_distribution(50000, 'Maxwell', kick_scaling=lambda x: (1-binary.m1_fallback_fraction)*x)

# Then we call the random draws
binary.get_natal_kick_distribution()

# Once the kicks are known, we derived the orbital parameters post kick: Porb, a, e
binary.get_orbital_distribution(True)

# Lastly, we obtain the grid of post-kick parameters
binary.get_post_kick_grid(use_unbounded_for_norm=True, verbose=True)
